In-Class Projects 📓

  • ClockIn

    ClockIn is a web application designed to make organization and task completion easier for users. This project was a continuation of the "Clock In" hackathon project but reimagined with a focus on organization and user experience. The site provides a space for users to dump all of their responsibilities and generate uuser tailored schedules and plans quickly. At ClockIn, we understand that true organization takes a long time, which takes away from valuable work time.


    I worked on this project in my XC475 fellowship and I served as the Product Owner. I worked with a team of developers and a designer to refine the project based on user feedback and typical college student workflows.

  • MediPet

    MediPet is a mobile application designed to help users with medicine adherence and tracking. The application features a user-friendly interface and provides reminders for medication schedules through the lens of a pet companion. Medications can be scheduled and tracked, and pets' moods change alongside user activity.


    I worked on this project in my CS411 course and I served as the Scrum Lead and Main Developer. Using React Native and the Expo framework, the project was developed with an object oriented design, making sure to focus on user experience and market fit.

  • Connect 4: Player vs. Lookahead-AI

    In my CS111 course, I used python to develop a program that allows users to simulate and play games of Connect-4! There are Player vs. Player, Player vs. AI and AI vs. AI modes, and the AI is fully incorporated to have a value known as "lookahead". The lookahead value allows for the AI to make predictions and calculations for their chance of winning and losing using recursive backtracking.

  • CBS Survivor Data Analysis: Prediction Modeling

    In my CS105 course, I used python and other libraries such as pandas, numpy, matplotlib, and sklearn to analyze data from the CBS Survivor TV show. I created a prediction model that predicts placements on the competition show based on the contestants' attributes and past performance. The model uses various machine learning algorithms to make predictions. I also created plots of placement on the basis of contestant attributes and created a presentation analyzing the results.
